# Chapter Resource Template

Use this template for resource chapters under `content/old-testament/...` or `content/new-testament/...`. These files sit beside the plain age-band Bible text and help the reader, future translators, teachers, and illustrators understand why the wording was chosen.

```markdown
# [Book] Chapter [Number]

## Book Overview
[One or two faithful sentences about this chapter's place in the book.]

## Verse-by-Verse Translation

### [Book] [Chapter]:[Verse]
**Original Reference**: [Book] [Chapter]:[Verse] - [Brief original-text meaning or traditional wording summary.]

#### Ages 5-7
[Rich, concrete, child-readable verse text.]

#### Ages 8-10
[Slightly fuller verse text with more biblical phrasing.]

**Translation Notes**:
[Explain theological, contextual, or wording choices. Use this when a child may ask, "Where did that come from?" or when later verses must not be imported too early.]

**Key Vocabulary**:
- [Word]: [Simple explanation]

**Cross-References**:
- [Related verse]

<!-- Illustration Prompt -->
**Scene**: [Visible moment from the verse]
**Style**: [Age-appropriate visual direction]
**Characters**: [Who appears, if anyone]
**Setting**: [Place and atmosphere]
**Action**: [What is happening]
**Emotional Tone**: [Wonder, courage, sorrow, joy, etc.]
**Age Appropriateness**: [Safety note]
<!-- End Illustration Prompt -->

---

## Chapter Summary
[Faithful, vivid summary. Do not turn theological claims into slogans that shift meaning.]

## Key Lessons for Children
1. **[Lesson]**: [Short explanation.]

## Memory Verses by Age

### Ages 5-7
[Short memory verse] - [Reference]

### Ages 8-10
[Longer memory verse] - [Reference]

## Discussion Questions by Age

### Ages 5-7
1. [Concrete question]

### Ages 8-10
1. [Observation or meaning question]

## Prayer
[Short prayer connected to the chapter.]
```

## Required Habits

- Keep the verse number structure exact.
- Keep difficult details when they belong to the verse; make them visible instead of deleting them.
- Add `Translation Notes` when wording could look like a contradiction with another chapter.
- Add `Key Vocabulary` for important keywords, unusual objects, theological words, or repeated book terms.
- Add `Cross-References` when another verse directly helps explain the passage.
